# BookManage
基于Flask+Vue的个人图书管理系统（本人毕业设计）


## 技术栈
- **后端**
: Python Flask + MySQL?sqlite
- **前端**
: Vue3 + Element Plus
- **工具链**
: Swagger API文档/Celery异步任务 ?

## 功能特性
- 登录/注册
- 图书管理
- 图书检索
- 图书分类


## 快速启动
```bash
# 后端
cd back
pip install -r requirements.txt
flask run

# 前端
cd test
npm install
npm run dev
```

## 测试启动
```bash
cd 项目根目录/
python back/app.py

cd test/
npm run serve
```
