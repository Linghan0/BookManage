# BookManage
基于Flask+Vue的图书管理系统（本人毕业设计）


## 技术栈
- **后端**
: Python Flask + MySQL
- **前端**
: Vue3 + Element Plus
- **工具链**
: Swagger API文档/Celery异步任务

## 功能特性
-
 × 扫码快速借还书
-
 × 借阅到期微信提醒
-
 × 多维度图书检索

## 快速启动
```bash
# 后端
cd back
pip install -r requirements.txt
flask run

# 前端
cd
 frontend
npm install
npm run dev
```
